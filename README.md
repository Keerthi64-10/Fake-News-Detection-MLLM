This project presents a Multi-Modal Large Language Model (MLLM) framework for automated news assessment and context-aware misinformation detection. It combines DistilBERT for text understanding and CLIP for image feature extraction with cross-attention fusion to analyze text, images, and metadata simultaneously. The system classifies news as REAL or FAKE while providing confidence scores and explainable fact-check insights. Built using Python, PyTorch, Hugging Face Transformers, and Streamlit, the framework aims to improve the accuracy, reliability, and transparency of fake news detection.
